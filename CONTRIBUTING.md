# Contributing

First of all, thank you for your interest in this project.

This repository was created to document a real firmware issue affecting Hikvision HWI-B140H cameras and to share a reliable workaround based on official Hikvision ISAPI commands.

## How you can contribute

Contributions are welcome, especially if you can:

- Test the workaround on different camera models.
- Test newer firmware releases.
- Confirm whether the bug is present or fixed.
- Improve the documentation.
- Report issues or unexpected behavior.
- Suggest cleaner or more efficient ISAPI implementations.

## Before opening an Issue

Please include as much information as possible:

- Camera model
- Firmware version
- NVR model (if applicable)
- Whether Virtual Host is enabled
- Steps to reproduce the issue
- Expected behavior
- Actual behavior

## Pull Requests

Please keep pull requests focused and limited to a single topic.

Examples:

- Documentation improvements
- Bug fixes
- Script improvements
- Additional compatibility testing

## Code Style

If modifying the workaround script:

- Keep the script compatible with standard Bash.
- Do not introduce unnecessary dependencies.
- Preserve compatibility with official Hikvision ISAPI commands.
- Clearly document any behavioral changes.

## Be respectful

Constructive feedback is always appreciated.

The purpose of this repository is to document the issue, help other users, and collect additional technical information.

Thank you for contributing!

---

If Hikvision releases a firmware update that permanently fixes this issue,
please open an Issue or Pull Request so the documentation can be updated for everyone.
